URL Shortener Microservice

User Story

  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • When I visit that shortened URL, it will redirect me to the original link.

Example creation usage:

https://utopian-music.glitch.me/new/https://www.google.com
https://utopian-music.glitch.me/new/http://foo.com:80

Example creation output:

{"original_url":"https://www.google.com","short_url":"http://utopian-music.glitch.me/SyQo2Nr2z"}

Usage:

http://utopian-music.glitch.me/SyQo2Nr2z

Redirects to:

https://www.google.com/
